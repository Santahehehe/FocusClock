前端:
1.當前端向某url路徑去做request請求時並不會跳轉頁面，除非下window.location()

2.let vs const:
	。let允許重新賦值 且 允許聲明時不賦值
	。const不允許重新賦值 且 聲明時必須賦值

3.localStorage.setItem("username",null);
  表示localStorage.getItem("username")的值是"null"而不是回傳null

4. .value適用在表單元素，innerText跟textContent可用來獲取<span>內容
	。.innerText 會返回元素的可見文本內容，並且會受到 CSS 樣式（如隱藏元素）的影響。
	。.textContent 會返回元素的所有文本內容，不會受到 CSS 的影響，並且包含所有子元素的文本。

5.url:用變數代入要用`而不是'，然後變數用${變數名稱}寫
	ex:`http://localhost:8080/save-time/${encodeURIComponent(userName)}`;

6.javascript中時間相減之結果單位是毫秒
  。javaScript將時間轉成字串後可以再轉回Date()形式

7.tagData 在 fetch 外面 console.log 顯示為空陣列
如之前所提，fetch 是異步的，所以在 fetch 完成之前，console.log(tagData) 就被執行了，此時 tagData 還沒有被填充。這是預期行為，因為 fetch 在運行中不會等待返回結果。

2.
在 Java 中，== 比较的是两个对象的引用是否指向同一块内存区域，而不是比较它们的内容。即使两个字符串的内容相同，它们可能指向不同的内存位置，因此 == 比较的结果为 false。
	解决方案
要比较两个字符串的内容是否相等，应该使用 String 类的 equals() 方法。equals() 方法会比较两个字符串的内容是否相同，而不是它们的引用是否相同。
